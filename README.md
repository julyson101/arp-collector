# ARP Collector

A Python-based network automation tool that connects to network devices via ssh and collects ARP table information

## Features
- SSH access using Netmiko
- YAML-based device inventory
- Easily extensible for more network tasks

## Requirements
- Python 3.8+
- Netmiko

## Setup

### 1. Close the repository
git close https://github.com/julyson101/arp-collector.git
cd arp-collector

## To run
go to project folder: /arp-collector/
activate virtual environment: source arpc-venv/bin/activate
run the code: python3 src/main.py
